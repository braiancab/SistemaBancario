# Sistema bancario
Nuestro proyecto trata de desarrollar un sistema bancario, utilizando una arquitectura monolitica en capas para mayor simplicidad inicial, facilidad de despliegue, depuración sencilla y bajo acoplamiento técnico inicial.


## Funcionalidades a desarrollar:
Nos centramos en la funcionalidad de transferencias a cuentas de terceros

## Tecnologías: 
- Backend: Utilizamos JAVA, junto con su framework Spring Boot
- Frontend: Utilizamos README y Bootstrap.
- Base de datos: MySql Workbench
